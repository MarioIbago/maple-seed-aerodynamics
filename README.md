Análisis Aerodinámico de Sámaras

Diseño Experimental
Variable Independiente

Altura de liberación ($h$). Es el factor que manipulamos directamente.

Variable Dependiente

Distancia horizontal ($d$). El resultado medido tras la caída.

Variables de Control

Viento, masa y ala. Parámetros constantes para asegurar validez.

Modelo Matemático
La velocidad terminal ($V_t$) determina el tiempo de vuelo. Semillas con alas más largas generan mayor sustentación, reduciendo $V_t$ y permitiendo mayor dispersión.

V_t ≈ 13 × √m / L_ala
Basado en datos de autorrotación real

¿Qué es una Sámara?
La sámara es un tipo de fruto seco indehiscente (que no se abre al madurar) provisto de un ala membranosa. Esta estructura es una adaptación evolutiva para la anemocoria: la dispersión a través del viento.

A diferencia de una caída libre, el ala de la sámara genera un par de fuerzas que induce una autorrotación estable. Este efecto de "helicóptero" reduce drásticamente la velocidad terminal, permitiendo que incluso brisas ligeras transporten la semilla lejos del árbol progenitor, reduciendo la competencia intraespecífica.
